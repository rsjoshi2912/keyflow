# KeyFlow — promotional site

Landing page for **KeyFlow**, a typing test & practice app for macOS and Windows.

- **Live site:** https://rsjoshi2912.github.io/keyflow/
- **Downloads:** attached to [GitHub Releases](https://github.com/rsjoshi2912/keyflow/releases) — Mac (Apple Silicon), Mac (Intel), and Windows (64-bit) installers.

The site is a single self-contained `index.html` (no build step) that reuses KeyFlow's own design tokens, served via GitHub Pages from the `main` branch.
