# KeyFlow — free typing software for Mac & Windows

**KeyFlow** is a free, open-source typing test & touch-typing practice app for macOS and Windows: timed typing speed tests with live WPM, a 10-step lesson course, and a progress dashboard. 100% free, offline-first, no ads, no account — all data stays on your device.

- **Website:** https://rsjoshi2912.github.io/keyflow/
- **Download (Mac Apple Silicon / Mac Intel / Windows):** https://github.com/rsjoshi2912/keyflow/releases/latest

## Features

- Typing speed tests (30s–10min) + passage mode with live WPM & accuracy
- Net & gross WPM, keystroke accuracy, consistency score — exam-grade formulas
- Learn touch typing: 10-lesson course from home row up, with accuracy gates
- Progress dashboard: speed trend, personal bests, daily streak
- Dark/light themes · fully offline · private · MIT licensed

This repository hosts the landing page (a single self-contained `index.html`, served by GitHub Pages from `main`) and the release binaries.
